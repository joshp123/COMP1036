Assignment 1

Content

  • Introduction

    Complete the following programs.  Each program carries 10 marks.  In each case some marks are available for elements of style (header blocks, commenting, meaningful names, well laid out code) as well as for functionality.

    Submit the source code (.py files) using the links at the bottom. Use the file-names given. 

  • 1. Your Initials

    Write a program that prints your initials in large characters made up of hash characters. The precise design of the letters is up to you. For the avoidance of any doubt, the program should print, below the initials, a blank line and then your name. For examples, if your initials were "ZSC", the output of your program should be something like:

      ###  ##  ##    # #   #     #   #  #    #     # #    ### ##   ##

    My name is Zacariah Stephen Cabinet.

    Remember to include a header block of comments in your program.

    Filename: initials.py

  • 2. The Canonical Second Program

    We have met the canonical first program (it prints "Hello World" on the screen). This is the canonical second program.Write a program that prompts the user to enter their name. Once they have done that, the program should display a cheery greeting. For example (here the user's name is "Zac"):

    Please Enter your name: ZacHello, Zac. I'm very pleased to meet you!The cheery greeting can be anything you like, as long as it appears on one line, as shown.

    Hint: Remember concatenation (addition) of strings.

    Filename: second.py

  • 3. The Problem of Eggs

    Eggs are traditionally sold in boxes of six. Write a program that prompts the user to enter the number of eggs they have in their basket. The program should display the number of egg boxes that can be filled (with six eggs), and the number of eggs that will be left over. For example three runs of the program might be:

    How many eggs are there in the basket? 25
    You need 4 boxes, and will have 1 egg left over.

    How many eggs are there in the basket? 3
    You cannot fill any boxes, and will have 3 eggs left over.

    How many eggs are there in the basket? 3.5
    Please enter an integer!

    For full marks, check that the value the user enters really is an integer, as shown in the third example. It should obviously also be a positive number!

    For a first class mark, print the results in a grammatically correct sentence, as shown!

    Filename: third.py
